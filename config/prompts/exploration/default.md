# 주제 탐색 프롬프트

다음 콘텐츠를 바탕으로 주제를 확장하고, 독자에게 유용한 인사이트를 도출해주세요.

**제목**: {{ title }}

**원본 요약**:
{{ summary }}

**핵심 포인트**:
{% for point in key_points %}
- {{ point }}
{% endfor %}

**태그**: {{ tags | join(", ") }}

---

다음 형식으로 분석해주세요:

[주제 확장]
- 이 주제를 더 넓게 보면 어떤 맥락이 보이나요?
- 관련 산업/기술/사회 트렌드는 무엇인가요?
- 이 주제가 향후 어떻게 발전할 수 있을까요?

[관련 논의와 반론]
- 이 주제에 대해 어떤 다른 관점이 있나요?
- 주요 찬반 논쟁 포인트는 무엇인가요?
- 비판적 시각에서 볼 때 어떤 점을 고려해야 하나요?

[독자 인사이트]
- 독자가 이 정보를 통해 무엇을 배울 수 있나요?
- 실무에 어떻게 적용할 수 있나요?
- 어떤 행동을 취하면 좋을까요?

[롱폼 작성 가이드]
- 가장 흥미로운 각도는 무엇인가요?
- 독자의 관심을 끌 수 있는 훅(Hook)은?
- 구조적으로 어떻게 전개하면 좋을까요?
