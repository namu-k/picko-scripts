```markdown
# 글 작성 프롬프트

## 1. 어조/톤 가이드
- authoritative: 신뢰감을 주는 전문적인 어조를 사용하세요.
- informative: 독자에게 유용한 정보를 전달하는 데 중점을 두세요.
- casual: 너무 딱딱하지 않게 친근한 느낌을 유지하세요.

## 2. 문장 구조 가이드
- 문장은 중간 길이로 균형 있게 구성하되, 복잡한 문장은 피하세요.

## 3. 글 구조 템플릿
1. **헤드라인**: 주제를 간결하게 요약하는 강렬한 제목 작성
2. **요약**: 주제에 대한 간단한 요약 제공
3. **상세 내용**: 주제에 대한 구체적인 정보와 데이터 제공
4. **인용**: 관련 전문가의 의견이나 인용 추가

## 4. 사용할 어휘/표현 스타일
- tech_terms: 기술 용어를 적절히 사용하세요.
- industry_jargon: 업계 전문 용어를 포함하세요.
- data_driven: 데이터 기반의 사실을 활용하세요.

## 5. 시작과 끝 패턴
- **시작 패턴**: "최신 개발 사항에 대한" 또는 "긴급 뉴스로" 시작하세요.
- **끝 패턴**: "요약하자면" 또는 "여러분은 어떻게 생각하시나요?"로 마무리하세요.
```
