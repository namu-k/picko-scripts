```jinja
# 영상 스크립트 프롬프트 템플릿

## 오프닝 훅
{{ hook }}
예: "최신 인공지능 기술이 산업에 미치는 영향에 대한 최신 소식을 전해드립니다."

## 본문 구성
### 헤드라인
{{ headline }}
예: "인공지능의 최신 발전"

### 요약
{{ summary }}
예: "이번 영상에서는 인공지능 기술의 최신 동향과 그 산업적 영향을 간략히 살펴보겠습니다."

### 세부사항
{{ details }}
- 기술 용어와 산업 전문 용어를 포함한 정보 제공
- 데이터 기반의 통계 및 사례 소개

### 인용
{{ quotes }}
예: "전문가들은 'AI는 앞으로 10년간 산업을 혁신할 것이다'라고 말합니다."

## 시각적 요소 가이드
- B-roll: 관련 기술의 사용 장면, 산업 현장
- 그래픽: 데이터 시각화, 인포그래픽

## 톤/목소리 가이드
- 톤: {{ tone }} (권위적, 정보 제공, 캐주얼)
- 목소리: 명확하고 자신감 있는 발음

## 클로징/CTA
{{ closing }}
예: "이 정보를 바탕으로 여러분은 어떻게 생각하시나요? 댓글로 의견을 남겨주세요!"
```
