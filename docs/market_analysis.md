# 시장 현황: 경쟁 구도 분석

**TL;DR**: 경쟁자 많지만, 정확히 "Picko와 같은 것"을 하는 상품은 없습니다. 다만 조각조각으로 경쟁합니다.

## 경쟁 상황의 진짜 모습

시장이 철저히 단편화되어 있습니다. 한 가지만 잘하는 도구들입니다.

```
RSS 리더 (Feedly) ────┐
                      ├─ 사용자 직접 조합
AI 글쓰기 (ChatGPT) ──┤  (번거로움)
                      ├─ 일부 스택: Zapier + ChatGPT + Buffer
소셜 스케줄러 (Buffer)┘   (비용 높음)
```

### 각 카테고리별 현황

| 카테고리 | 주요 경쟁사 | 가격 | 한계 |
|----------|-------------|------|------|
| RSS 리더 | Feedly, Inoreader | 무료~$100/월 | 콘텐츠 생성 불가 |
| AI 글쓰기 | ChatGPT, Jasper, Copy.ai | $20~$300/월 | RSS 연동 없음 |
| SNS 스케줄러 | Buffer, Later, Hootsuite | $12~$500/월 | 생성 기능 약함 |
| 자동화 플랫폼 | n8n, Zapier, Make.com | 무료~$600/월 | 일반적(콘텐츠 특화 X) |
| 뉴스레터 | Beehiiv, Substack | $0~$200/월 | RSS 발굴 미약함 |

## Picko의 실제 경쟁 상황

### ① 존재하지 않는 경쟁자

"RSS → 채점 → 큐레이션 → 다중채널 생성 → 렌더링"을 한 번에 하는 상품은 없습니다.

**가장 가까운 것:**
- n8n + Zapier: 가능하지만 각각 기술 설정 필요 (코딩 수준), 월 비용 $100+
- Jasper + Buffer + Feedly 스택: 각각 따로 구매하면 월 $100+ + 데이터 연동 번거로움

### ② 있는 경쟁자

#### a) 보편적 선택: "개별 도구 조합"

현실: 현재 콘텐츠 마케터 대부분이 이렇게 합니다.

```
Feedly(RSS 읽기) + ChatGPT(글쓰기) + Buffer(스케줄) + DALL-E(이미지)
```

- 비용: 월 $80~120
- 시간: 여전히 하루 2~3시간

**Picko의 우위:**
- 한 달에 한 번만 세팅 후 매일 5분
- 월 $50 이하 (로컬 AI 활용)

**문제:**
- 사용자가 이미 개별 도구에 익숙함
- 전환 비용(switching cost) 높음

### ③ 강적: n8n

n8n은 실은 Picko의 가장 강력한 경쟁자입니다.

n8n + OpenAI API로 같은 것을 만들 수 있습니다:
1. RSS 트리거
2. OpenAI API로 요약·태그
3. 여러 채널에 발행
4. 데이터 저장소 연동

**가격:**
- 자체 호스팅: 무료
- 클라우드: $20~$600/월

**문제:**
- 초기 세팅에 기술 필요 (workflow 디자인)
- 각 단계 개별 설정 (자동화 X)
- 콘텐츠 마케팅 특화가 아님 (generic automation)

하지만 n8n 선택 비율: 개발팀이 있는 기업 10% 수준. 개인사업자는 거의 안 씀.

## Picko의 경쟁력이 있는 이유

### 1. 니치 시장의 왕

일반 AI 도구 회사들은 Picko를 경쟁 대상으로 생각하지 않습니다. (너무 specific)

- ChatGPT의 입장: "우리는 일반 AI, 콘텐츠 파이프라인 특화는 관심 없음"
- Buffer의 입장: "우리는 SNS 스케줄러, RSS 기반 발굴은 관심 없음"
- n8n의 입장: "우리는 범용 automation, 콘텐츠 UI는 만들 수 없음"

**결론:** 직접 경쟁자가 정의상 없습니다. (또는 극도로 분산됨)

### 2. 개인사업자 특화 구조

대부분의 경쟁자는 팀 크기 10명 이상인 기업을 타겟합니다. (월 $100+ 가격대)

**Picko의 가치 제안:**
- 1인 미디어 운영자: 월 매출 $3,000~10,000 정도인 사람
- 개인 블로그/유튜브/뉴스레터: "콘텐츠는 중요한데 시간이 없는" 사람
- 코치/컨설턴트: "신뢰도를 위해 꾸준히 콘텐츠 배포하고 싶은" 사람

→ 이 그룹은 Jasper($50/월)나 Buffer($15/월) 하나를 선택하지, 3개를 조합하지 않습니다.

### 3. 로컬 AI 아키텍처의 장점

다른 경쟁사들은 모두 클라우드 API 의존:
- ChatGPT: $20/월 고정 + 오버리미트 시 문제
- Jasper: 생성량 기반 과금
- Buffer AI: 월 수십 크레딧 제한

**Picko의 장점:**
```
로컬 LLM (Ollama) → 요약·태그·임베딩
    ↓
무제한 호출, $0 비용

클라우드 LLM → 실제 글쓰기만
    ↓
월 $10~20 정도면 충분
```

월 $50 vs 월 $150의 차이 = 개인사업자 입장에서 매달 $100의 절감

## 진짜 문제점 (솔직한 평가)

### 1. 블러딩 아티팩트의 위험

AI 도구 시장에서 새로운 기능이 매우 빠르게 흡수됩니다.

- 2024년: "Feedly에 AI 없음" → Feedly가 AI 추가
- 2025년: "Buffer에 AI 없음" → Buffer가 AI 추가
- 2026년: "Zapier에 AI 없음" → Zapier가 AI 추가

**2027년에는?**
- "Obsidian에 RSS 발굴·생성 없음" → Obsidian이 추가할 가능성

Picko의 장점이 점차 무디어질 수 있습니다.

### 2. UI/UX 진입 장벽

현재 Picko는 CLI 기반입니다. 일반인이 터미널 명령어를 매일 입력하는 건 현실적이지 않습니다.

- 개인사업자 입장: "커멘드 라인? 아, 이건 나 아니다" → 이탈
- 경쟁사들(Buffer, Later, Jasper) 모두 웹 UI가 있습니다.

### 3. 온보딩 복잡도

첫 세팅:
1. Python 3.13 설치
2. Ollama 설치 + 모델 다운 (시간 걸림)
3. 가상환경 설정
4. YAML 파일 커스터마이징
5. API 키 설정

→ 기술 능력 없는 사람은 포기합니다.

경쟁사들은 웹에서 "계정 만들기 → API 키 입력 → 끝" (5분)

### 4. 검증 필요

"실제로 ROI가 나는가"를 증명해야 합니다.

- Jasper: "마케터 500명이 월 $50 지불" (검증됨)
- Buffer: "50만 사용자" (검증됨)
- Picko: 아직 검증 사례 없음

## 시장 진입 전략 추천

### 방안 A: 틈새시장 전략 (추천)

**타겟:** 기술에 익숙한 개인사업자 (개발자, 퍼블리셔, 기술 블로거)

**시작:**
1. Reddit, Product Hunt, Hacker News에 자신감 있게 진출
2. "CLI 기반 RSS → AI → 멀티채널" 포지셔닝
3. 초기 사용자 100명 확보 (word-of-mouth)
4. 성공 사례 수집 → case study 제작

**요구사항:**
- 웹 UI 없음 (CLI만)
- 설정 한 번에 30분 투자 요청
- 회비: 월 $20~30 (저가 포지셔닝)

**경쟁사:** 없음 (이 틈새는 경쟁사가 신경 안 씀)
**성장 예상:** 느리지만 안정적

### 방안 B: 웹 UI 추가 후 광범위 공략

**시작:**
1. 웹 UI 완성 (6-12개월 투자)
2. Buffer/Later 수준의 UX 달성
3. 광고 시작 (Product Hunt, Twitter, TikTok)
4. 가격: 월 $29~79 (중가 포지셔닝)

**경쟁사:** Buffer, Later, Ocoya, FeedHive
**어려움:** 이들이 이미 50만+ 사용자 보유
**성공 조건:** 뚜렷한 차별점 필요 (예: "가장 저렴한 멀티채널 AI", "가장 정확한 RSS 관련성 채점")

### 방안 C: B2B2C (에이전시 판매)

**타겟:** 소규모 마케팅 에이전시 (5-20명)

**가치:**
- 에이전시가 클라이언트 마다 자동화 구축 → 시간 절감
- 클라이언트당 월 $50 × 10 클라이언트 = 월 $500 매출

**판매 방식:**
1. 5개 에이전시에 직접 피치
2. 성공 케이스 확보
3. 연쇄 판매

**초기 난이도:** 낮음
**성장 속도:** 중간 (입소문)

## 결론

경쟁자는 많지만, 당신이 하는 정확한 일을 하는 경쟁자는 없습니다.

**BUT:**
1. 더 강력한 일반 도구들이 점점 콘텐츠 마케팅으로 진출 중
2. 웹 UI 없으면 대중 시장 진입 불가
3. 기술 장벽이 높아서 초기 사용자 확보 어려움
4. 검증된 사용 사례가 필요함

**추천:**
- **Stage 1 (3개월):** 기술자 커뮤니티 타겟, CLI 기반, 월 $20, 초기 100사용자 확보
- **Stage 2 (6-12개월):** 피드백 수집 → 웹 UI 개발 → 가격 인상 → $29~79 대중 시장 진출
- **Stage 3 (18개월+):** B2B 에이전시 영업, 엔터프라이즈 패키지

시장은 커지는 중입니다. 남은 건 실행력과 타이밍입니다.
