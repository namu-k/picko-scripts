# 005-orchestration-layer

콘텐츠 파이프라인 오케스트레이션 레이어 — Vault 상태 기반 워크플로우 자동화

## 개요

기존 스크립트들(`daily_collector.py`, `generate_content.py`, `render_media.py`)을
**선언적 워크플로우**로 연결하는 오케스트레이션 레이어.

1. **Vault 상태 어댑터**: frontmatter 필드를 조건으로 사용하는 쿼리 인터페이스
2. **배치 처리**: 대량 아이템을 크기/딜레이 제어하며 처리
3. **승인 대기/재개**: Vault 상태 변경을 감지하여 다음 단계 실행
4. **임베딩 액션**: 중복 검사를 워크플로우 step으로 선언

## 설계 원칙

- Vault를 유일한 상태 저장소로 유지 (새 DB 없음)
- 기존 스크립트를 액션으로 래핑 (점진적 마이그레이션)
- 전체 DSL을 설계하되, 1차 구현은 핵심 기능만
