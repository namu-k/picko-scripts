## 이 방식은 '자동화 고수'들이 가장 선호하는 **'로컬 기반 AI 워크플로우'**입니다.

## 이 구상하신 방식은 **"내 컴퓨터를 AI의 작업실로 만들고, 클로드 코드를 비서로 쓰는 것"**과 같습니다. 이 시스템을 구축하기 위한 핵심 단계를 정리해 드릴게요.

---

### 1. 1단계: 로컬 폴더에 자동으로 저장하기 (수집 자동화)

가장 먼저 RSS와 퍼플렉시티 결과가 내 폴더에 자동으로 '파일'로 떨어지게 만들어야 합니다.

* **RSS 피드:** `Python` 스크립트를 하나 짜서(클로드에게 짜달라고 하면 1분 걸립니다), 특정 시간마다 RSS를 긁어 `.md`(마크다운) 파일로 저장하게 합니다.
* **Perplexity Tasks:** 퍼플렉시티 태스크는 이메일로 결과를 보내줍니다. **Make**나 **Zapier**를 사용해서 "이메일이 오면 내용을 추출해서 내 드롭박스(Dropbox)나 구글 드라이브 폴더에 `.txt`로 저장해줘"라고 설정합니다.
* **로컬 동기화:** 드롭박스/구글 드라이브 로컬 동기화 기능을 쓰면, 구름 위에 저장된 파일이 즉시 내 컴퓨터 폴더로 내려옵니다.

구체 구현:
- RSS 수집 스크립트: `C:\picko-scripts\scripts\simple_rss_collector.py`
- Perplexity 이메일 → 파일 저장 설정 가이드: `C:\picko-scripts\mock_vault\config\Folders_to_operate_social-media_copied_from_Vault\0. 프레임워크\Perplexity_email_setup.md`

### 2. 2단계: 클로드 코드(Claude Code)에게 일 시키기

이제 해당 폴더에서 터미널을 열고 `claude`를 실행합니다. 클로드 코드는 자신이 실행된 폴더 내의 모든 파일을 읽을 수 있는 권한이 있습니다.

> **클로드 코드에게 던지는 명령 예시:**
> "이 폴더에 새로 생긴 RSS 파일들이랑 퍼플렉시티 리포트 읽어봐. 겹치는 내용 제외하고, 오늘 가장 중요한 테크 뉴스 3가지만 뽑아서 `daily_report.md` 파일로 정리해줘."

### 3. 이 방식의 압도적인 장점

* **클로드 API 비용 절감:** 클로드 코드는 보통 클로드 프로(Pro) 구독 범위 내에서 사용하거나 별도의 정액제 기반이라, 호출마다 돈이 나가는 API보다 대량 작업에 유리할 수 있습니다.
* **히스토리 관리:** 모든 레퍼런스가 내 컴퓨터에 파일로 남기 때문에 나중에 검색(grep 등)하기가 매우 좋습니다.
* **커스터마이징:** 클로드 코드에게 "내 블로그 톤앤매너가 담긴 `STYLE.md` 파일을 참고해서 글을 써줘"라고 하면, 매번 똑같은 지시를 할 필요가 없습니다.

### 🛠️ 사용자님을 위한 시스템 설계도

1. **폴더 생성:** `C:/MyAIWorker/inputs` (수집 폴더), `C:/MyAIWorker/outputs` (결과물 폴더)
2. **수집기:** * **RSS:** Python 스크립트 (Windows 작업 스케줄러로 매일 실행)
* **Perplexity:** Zapier/Make → 이메일 수신 → 로컬 동기화 폴더에 저장


3. **처리자:** * 터미널에서 `cd C:/MyAIWorker`
* `claude "inputs 폴더의 새 파일들 읽고 요약해줘"` 실행
